                                                                                              Roman Grinyov вопрос • 2 года назад

                                                                                              Селекторы здесь вообще ни при чём ...
                                                                                              Вы подключаете шрифт: 

                                                                                              @font-face {
                                                                                                  font-family: "FontName";
                                                                                                  src: url("path");
                                                                                              }

                                                                                              FontName — имя шрифта, которое будет использоваться далее в файле стилей. Его можно определить любым, хоть «Super Duper Pooper Font» (если имя шрифта не содержит пробелом, то кавычки (") можно опустить).
                                                                                              path — путь к шрифту. Шрифт можно загружать непосредственно с локального(!) (здесь имеется в виду с сервера, где расположен сайт, то есть путь к шрифту в «папке сайта») или сетевого источника (например, https://fonts.googleapis.co... , то есть Google Fonts).

                                                                                              Затем его используете: 

                                                                                              selector {font-family: "Super Duper Pooper Font";}
